## Installation

Add these lines to your Chat

	require 'namaste'

	Robut::Plugin.plugins << Robut::Plugin::Namaste



And then execute:

    $ bundle

Or install it yourself as:

    $ gem install robut-namaste

## Usage

@robut namaste me - returns quote
